# Build Recipe Use Case

## Success Scenario
1. User wants to build a recipe.
2. User writes the quantity of persons that will eat the recipe.
3. User may writes the ingredients of the recipe.
4. User may writes the kind of meal that the recipe is.
5. User click on the button "Build Recipe".
6. System shows the recipe built.

## Exception - invalid URL
1. System shows a message of unspected error.

## Exception - invalid data
1. System shows a message of unspected error.

## Exception - Server error
1. System shows a message of unspected error.
